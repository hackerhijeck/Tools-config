## Installing:
```
sudo apt-get update
mkdir ~/tmp
cd ~/tmp
wget https://www.python.org/ftp/python/3.9.0/Python-3.9.0.tgz
tar -xvzf Python-3.9.0.tgz
cd Python-3.9.0
./configure
sudo make install
```

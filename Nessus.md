### Nessus Installing:
```
1. Go to tenable site and register a account using tem mail. and copy your activation code.
2. Download the Nessus from official site.
3. Go to download folder in terminal:
4.  $ dpkg -i nessesfile
5.  $ systemctl start nessusd
6. Open with your browser IP:port
```
### Nessus Uninstalling:
```
1. Go to opt folder:
   $ cd /opt
2. Delete Nessus file:
   $ rm -rf Nessus
3. Again follow Installing steps.
```
